# Singly Linked List in Java

This project is a simple implementation of a **Singly Linked List** in Java.
The program allows the user to add elements to the end of the list using a separate `main` class.

## Features
- Singly linked list implementation
- Add element to the end of the list
- User input using `Scanner`
- Print all elements of the list

## Project Structure

SinglyLinkedList/
│
├── SinglyLinkedList.java
├── Main.java
└── README.md

## How to Run
1. Open the project in **NetBeans**
2. Run `Main.java`
3. Enter the number of elements
4. Enter each element when prompted

## Sample Output

Enter element 1: 10
Enter element 2: 20
Enter element 3: 30
10 -> 20 -> 30 -> null
